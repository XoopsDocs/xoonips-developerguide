## 1\. array xnpGetAttachmentDetailBlock( int item_id, string name ) {#1-array-xnpgetattachmentdetailblock-int-item-id-string-name}

アイテム詳細画面に表示するフォームを作成します．ファイル名とサイズを表示します．ファイルをダウンロードするにはファイル名をクリックします．
